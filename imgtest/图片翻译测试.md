## English paragraph for translation test

This is a Markdown viewer built with WPF and Prism.
It can render Markdown files with code highlighting and a table of contents.
The translation feature converts English text into Simplified Chinese.

- First feature: open and render markdown
- Second feature: recent files and favorites
- Third feature: dark theme and image support
